# Wheel-Leg-Robot-Control: PID, LQR and MPC Implementation

## 1. 项目简介 | Project Overview
本项目针对轮腿机器人（Wheel-Legged Robot）的平衡与运动控制，完成了从动力学建模到高阶控制算法设计的全流程开发。本项目不仅实现了基础的 PID 控制，还深入探讨了状态空间控制（LQR）以及现代控制理论中的模型预测控制（MPC）。
通过本项目，验证了不同控制策略在处理非线性、强耦合系统时的性能差异，特别是在输入约束（Input Constraints）下的系统稳定性。

## 2. 核心算法设计 | Core Algorithm Design
本项目核心逻辑基于拉格朗日方程建立的动力学模型，涵盖了以下三种控制策略：
### A. PID 控制 (基础反馈)
  实现逻辑：将系统解耦为平衡、前进、转向三个子回路
  特点：简单易行，但在处理多变量耦合及阶跃响应时存在明显的超调。
### B. LQR (线性二次型调节器)
  实现逻辑：通过构建状态空间模型，求解代数黎卡提方程（ARE）获得最优反馈增益。
  优势：调节速度快（<1.5s），实现了多变量的最优加权控制。
### C. MPC (模型预测控制) —— 核心亮点
  实现逻辑：将控制问题转化为有限时域内的二次规划（QP）问题。
  创新点：约束处理，严格限制电机力矩输入，模拟真实硬件的物理极限。参考轨迹生成，在当前状态与平衡位置间生成动态线性参考向量。在满足物理约束的前提下，实现了比 LQR 更稳健的动态响应。
